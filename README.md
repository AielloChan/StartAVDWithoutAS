# How to use

1. Download "startAVD"
2. Add excute permission by `chmod +x startAVD`
3. Configure emulator path: `vim startAVD` and change `emulatorPath` to your emulator location
4. Run It! `./startAVD`

```bash
./startAVD 
> Select emulator to start:
> 	 0)	 Nexus_S_API_27
> 	 1)	 Pixel_2_API_17
> 
> Input index: 
```

Thats it!
